# vim cheetsheet


## basic operation

### move tab
- '<tab>h'	: move tab left
- '<tab>l'	: move tab right

### move
- '[num]gg'	: move to [num]th line
- 'G'				: move to end
- '0'				: move to line head
- '$'				: move to line end

### related search
**search move**
- 'n'				: forward direction
- 'N'				: backward direction

**1line 1char**
- 'f'				: 1line 1char forward
- 'F'				: 1line 1char backward
- ';'				: re 'f'
- ','				: re 'F'

**brackets move**
- '%'				" move to pair brachet
